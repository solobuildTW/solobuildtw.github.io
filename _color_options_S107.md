# 8 套網頁配色風格選擇（S107 調研，2026-04-27）

> 來源：WebFetch 實際抓取各官網 HTML/CSS，非靠記憶。
> 定位需求：科技 + AI + 休閒，Harvey 工作室 / SoloBuildTW

---

## 風格 A：Vercel（純黑白極簡）

- URL：https://vercel.com
- 主底：`#000000` / 次底 `#0A0A0A`
- 卡片：`rgba(255,255,255,0.05)` + 細白邊框 1px
- 文字：`#FFFFFF` 主標 / `#888888` 副文字
- 強調：`#FFFFFF`（純白按鈕）/ Hero 漸層 cyan→magenta（細節點綴）
- 字體：Geist（自家設計）/ 次選 Inter
- 風格：極簡工程美學，大字重視覺衝擊，幾乎不用彩色
- 截圖描述：純黑底，超大白字 Hero，卡片用極淡玻璃邊框，Nav 透明毛玻璃效果，幾乎所有元素只有黑白兩色，唯一亮點是 Logo 或按鈕邊緣的微漸層
- 適合 Harvey？：科技感爆表，但休閒感幾乎為零。若 Harvey 主打硬核技術形象可考慮，工作室展示偏冷

---

## 風格 B：Linear（深紫黑玻璃擬態）

- URL：https://linear.app
- 主底：`#08090A`（極深黑）/ 次底 `#0F1011`
- 卡片：`#1C1C1F` + 邊框 `#23252A` / 半透明 `rgba(255,255,255,0.05)`
- 文字：`#FFFFFF` 主 / `#A0A0A0` 次
- 強調：`#5E6AD2`（靛紫）/ 漸層 `#5E6AD2→#4EA7FC`（藍靛）
- 輔助色：`#4EA7FC`（藍）/ `#68CC58`（綠）/ `#FC7840`（橘）
- 字體：Inter Variable
- 風格：深色系產品工具感，玻璃擬態卡片，色彩克制但有個性
- 截圖描述：接近黑底，Nav 半透明，Hero 文字帶細微紫色光暈，卡片有玻璃質感邊框，局部出現靛紫漸層點亮頁面，整體神秘且精緻
- 適合 Harvey？：非常適合 AI + 工具定位，有高端感，休閒度中等，可加暖色點綴提升親切感

---

## 風格 C：Anthropic（米白暖橘 Serif）

- URL：https://anthropic.com
- 主底：`#F0EEE6`（米白/oat）/ 次底 `#E8E6DC`（cream）
- 卡片：`#FFFFFF` / `rgba(250,249,240,0.5)`
- 文字：`#141413`（近黑）/ `#3D3D3A` 次 / `#87867F` 灰
- 強調：`#D97757`（暖橘）/ `#CC785C`（磚紅橘）
- 漸層：`rgba(204,120,92,0.5)` 橘色光暈
- 字體：Serif 標題 + Sans-serif 內文
- 風格：溫暖知性，學術感，米白底給人安全可信賴感，暖橘點綴不搶戲
- 截圖描述：整頁米黃暖白底，標題用 Serif 字型，橘色 CTA 按鈕，卡片無邊框靠色差區分，照片用暖色濾鏡，整體像高端雜誌而非科技產品
- 適合 Harvey？：休閒度最高，信任感強，但科技感較弱。適合「有溫度的 AI 工作室」定位，不適合純技術展示

---

## 風格 D：Stripe（漸層藍紫深邃）

- URL：https://stripe.com
- 主底：`#031323`（深海藍）/ `#0A1628`
- 卡片：`#061B31` + 藍邊框 `#0052FF` 細線
- 文字：`#FFFFFF` / `#8CA0B9` 次
- 強調：`#0096FF`（亮藍）/ `#0052FF`（電藍）
- 漸層：`#003478→#0096FF→#00DCFF`（藍色深海漸層）/ Hero 帶彩虹光暈
- 字體：Inter / Camphor
- 風格：金融科技感，深藍色建立信任感，漸層製造科技未來感，色彩豐富但不混亂
- 截圖描述：深海藍底，Hero 有大型 3D 彩虹漸層光球，卡片有微藍色內發光邊框，滾動時有視差漸變效果，整體感覺「有錢有品」
- 適合 Harvey？：適合有付費產品的工作室，信任感 + 科技感俱佳，但休閒感稍低，橘紅點綴可補

---

## 風格 E：Cursor（純黑細邊超極簡）

- URL：https://cursor.sh
- 主底：`#000000` / `#0A0A0A`
- 卡片：黑底 + 極細白邊框 `rgba(255,255,255,0.08)` / 近乎隱形
- 文字：`#FFFFFF` / `#6B6B6B` 弱化
- 強調：白色 + 偶爾亮藍 `#52AEFF`
- 漸層：幾乎無漸層，強調邊框細節而非色彩
- 字體：Geist / Inter
- 風格：比 Vercel 更硬核，近乎苦行極簡，細節控的天堂，只給開發者看
- 截圖描述：全黑頁面，文字小而密，卡片邊框細如髮絲，Hero 沒有大漸層只有細緻紋路，功能展示用 code snippet 截圖
- 適合 Harvey？：若主受眾是開發者可考慮，但對 Podcast / 經驗包受眾太冷，親切感接近零

---

## 風格 F：GitHub（中性灰白藍綠）

- URL：https://github.com
- 主底：`#0D1117`（深灰黑）/ `#161B22`（次）
- 卡片：`#1E2327` / `#24292F`
- 文字：`#E6EDF3` 主 / `#979A9C` 次
- 強調：`#238636`（綠）/ `#1F6FEB`（藍）
- 漸層：幾乎不用，純色為主
- 字體：-apple-system / Segoe UI 系統字
- 風格：低調中性，實用工具感，不搶眼但也不醜，長時間看不累
- 截圖描述：深灰底，綠色 CTA（Follow/Star），藍色連結，Nav 深灰，整體是工具而非品牌展示頁
- 適合 Harvey？：適合當「基礎款」，不會出錯但也沒亮點，若要做出工作室個人品牌感，需要更強烈的配色特徵

---

## 風格 G：OpenAI（白底黑字極致克制）

- URL：https://openai.com
- 主底：`#FFFFFF`（白）
- 卡片：`#F7F7F7` 淡灰 / `#EFEFEF`
- 文字：`#0D0D0D` 主 / `#343541` 次 / `#8E8EA0` 弱
- 強調：`#10A37F`（翠綠）/ `#ACACBE` 輔
- 漸層：極少，Hero 用微妙的 AI 生成視覺而非漸層
- 字體：Söhne（Klim Type Foundry）/ Inter
- 風格：最乾淨的白底，像蘋果官網的 AI 版，每個元素呼吸感極強
- 截圖描述：全白頁面，黑色大標題，翠綠 Logo，卡片無邊框靠大量留白區分，唯一的色彩來自 AI 生成的影片截圖
- 適合 Harvey？：留白很難駕馭，需要高品質圖像填補，若圖像資產夠強可考慮；休閒感中等，需加個人化元素

## 風格 H：daisyUI / Tailwind（暗色系橘紅漸層）

- URL：https://daisyui.com
- 主底：`#091427`（深海軍藍）/ 次底 `#0A0A0A`
- 卡片：`rgba(255,255,255,0.05)` + 彩色邊框漸層
- 文字：`#FFFFFF` / `#94A3B8` 次
- 強調：`#38BDF8`（天藍）/ `#C9F180`（螢光綠）/ 漸層 `#38BDF8→#818CF8`
- 漸層：大量色彩主題預設，可切橘/藍/紫/綠，demo 頁同時展示多套
- 字體：Inter / 系統字
- 風格：展示型、活潑、多彩，技術+設計感兼顧，比前幾套更「有趣」
- 截圖描述：深藍底，彩色 Badge 系統，卡片用彩虹漸層邊框，右側有即時主題切換器，整體像設計系統的遊樂場
- 適合 Harvey？：彈性最高，可調出適合工作室的子主題，活潑感與科技感並存，稍加克制即可落地

## 研究員推薦

**首選：風格 B（Linear 深紫黑）**
理由：科技感最強同時有品牌個性，靛紫 `#5E6AD2` 作強調色獨特且不俗，配色體系完整，四卡片可用不同強調色（紫/藍/綠/橘）對應 Harvey 四個主線產品（Podcast/經驗包/API工具/工作室），落地難度中等，最能在 AI 工作室定位中脫穎而出。

**次選：風格 C（Anthropic 米白橘）**
理由：若 Harvey 想要強調「有溫度的一人公司」而非純技術工具，米白+暖橘最親切，適合 Podcast 受眾，與競品的「黑底科技」做差異化定位。

**折衷方案：B + C 混合**
深色 Hero section（風格 B）+ 白色/米白內容 section（風格 C）+ 暖橘 CTA，可同時兼顧科技感與休閒感，但需要設計稿確認視覺一致性。
