FAQ

商家常問

我不會技術可以用嗎？

完全可以！全程不需要寫程式 / 不需要寫 prompt。後台 4 頁籤就是給沒技術背景的店家設計的 — 上傳檔案 + 用中文口頭教，就跟員工說話一樣。30 分鐘可上手。

我自己的 LINE OA 怎麼接？

商家給我們 LINE OA 的 Channel Secret 跟 Access Token（在 LINE Developers 後台 5 分鐘可拿），我們 30 分鐘內接好。或我們有線上/視訊 onboarding 服務（首月免費）。

我的店適合嗎？

所有的店都適合：餐廳 / 飲料店 / 燒烤攤 / 美容美甲 / 寵物美容 / 補習班 / 診所 / 工作室 / 零售小店 / 民宿 / 健身房 / 攝影工作室 / 服飾店 / 烘焙坊 / B2B 廠商等。只要您的店有客人會在 LINE 問問題，就值得裝一個 24 小時不下班的 AI 客服。客人少的店更划算——每筆對話都不會錯過。

客人會知道是 AI 嗎？

預設立場：不偽裝真人。被客人問「你是 AI 嗎」會直接說明自己是 Line 自動回覆機器人。回覆口語化自然，但不假裝真人接洽。

客人投訴會怎麼處理？

BOT 偵測到「投訴」「退費」「我要找真人」「敏感詞」自動立刻轉商家自己的 LINE，BOT 不會擅自處理。

可以做預約嗎？

可以。客人在 LINE 說「我要訂位 明天 6 點 4 人」→ BOT 自動建預約 → 通知商家 LINE → 商家確認後 BOT 回客人。進階版（T1）串 Google Calendar / Cal.com。

對話有上限嗎？

月費方案的對話量額度遠超過絕大多數小店一年用得到的量，正常店家完全用不完。萬一接到爆量檔期（連假/節慶/活動），會自動通知商家、提供加值方案，不會中斷服務。

用什麼 AI 模型？

DeepSeek V3，中文最強、台灣商家最划算。商家也可升級企業客製方案接 GPT-4 / Claude。

我的客戶資料安全嗎？

Supabase PostgreSQL + Row Level Security，每商家資料完全隔離。資料儲存在 ap-southeast-1（新加坡）。GDPR / 個資法相容。

月費 7 天內、半年費 / 年費 30 天內可退剩餘金額。14 天免費試用期不滿意直接停，不扣款。

FB / IG 也能用嗎？

FB Messenger Bot 規劃中（Roadmap Q3 上線）。IG 規劃中，預計在 Meta DM API 政策明朗化後第二波支援。目前先把 Line 體驗做到 100% 完整。
