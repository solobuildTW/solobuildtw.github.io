# 拿 5 個成交量最大的台股
curl -H "X-API-Key: YOUR_API_KEY" \
  https://taiwan-data-api.harvey3630.workers.dev/api/v1/stock | jq '.data.records[0:5]'

# 反查台北 101 在哪一行政區
curl -H "X-API-Key: YOUR_API_KEY" \
  "https://taiwan-data-api.harvey3630.workers.dev/api/v1/address?lat=25.033964&lon=121.565418"
import requests

API_KEY = "YOUR_API_KEY"
BASE = "https://taiwan-data-api.harvey3630.workers.dev"
headers = {"X-API-Key": API_KEY}

# 拿台北市實價登錄
r = requests.get(f"{BASE}/api/v1/realestate", params={"county": "臺北市"}, headers=headers)
data = r.json()["data"]
print(f"{data['count']} 筆，第一筆：{data['records'][0]}")
const API_KEY = 'YOUR_API_KEY';
const BASE = 'https://taiwan-data-api.harvey3630.workers.dev';

// 拿三大法人買賣超
const r = await fetch(`${BASE}/api/v1/institutional`, {
  headers: { 'X-API-Key': API_KEY }
});
const { data } = await r.json();
console.log(`${data.total_records} 個股法人籌碼，日期 ${data.date}`);

常見問題

資料來源是什麼？商用合法嗎？
全部來自台灣政府開放 API（TWSE、CWA、EPA、BOT、NLSC、內政部），授權為 OGDL 1.0 / CC BY 4.0，商業使用合法，建議標示來源。
跟 FinMind / Fugle 差在哪？
FinMind 強在台股深度，但不在 RapidAPI 生態圈、英文文件少；Fugle 即時 tick 但月費 NT$2,500 起。我們強在「一站式」整合 8 個 API，台股 + 氣象 + 地址 + 實價登錄 + 招標 + 法人，月費 NT$300 起，繁中文件給台灣開發者。
付款方式？台灣或海外？
台灣用戶：藍新金流（pay.solobuildtw.com），支援信用卡 / Apple Pay / Google Pay。海外用戶：RapidAPI Marketplace（解信用卡）。
年費跟月費差多少？
年費 17% 折扣（等於買 10 個月送 2 個月）。Pro 年費 NT$2,999（折合月費 NT$250）。
怎麼取消訂閱？
月費：到期不續即可（不會自動扣款）。年費：寫信至 solobuildtw@gmail.com，當期未用完部分按比例退費。
quota 用完會怎樣？
回 429 錯誤。Free / Pro 每日 0:00 UTC 重置。Ultra / Mega 無 quota 限制。
政府 API 改版了 API 還能用嗎？
我們有監控告警，政府 API 改版會在 24 小時內修復。改版期間 KV 緩存有 stale fallback，最壞回上一小時的資料。
支援自訂部署？私有雲？
Mega 方案以上提供，請聯繫 solobuildtw@gmail.com 客製方案。
