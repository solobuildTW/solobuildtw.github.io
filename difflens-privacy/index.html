DiffLens Privacy Policy

Last updated: April 28, 2026

What DiffLens Does

DiffLens is a browser extension that provides AI-powered code review for GitHub Pull Requests. It reads diff content from GitHub PR pages and sends it to an AI service for analysis.

Data We Collect

DiffLens collects and processes the following data:

  • Diff content from GitHub PRs — Sent to the AI backend for review. Not stored after processing.
  • API key — Stored locally in your browser using chrome.storage.local. Never transmitted to our servers.
  • License key — Sent to Lemon Squeezy for validation only. Not stored on our servers.
  • Daily usage count — Stored locally to enforce free tier limits. Not transmitted anywhere.

Data We Do NOT Collect

  • No personal information (name, email, location)
  • No browsing history
  • No GitHub credentials or tokens
  • No analytics or tracking (no Google Analytics, no Sentry)
  • No cookies

How Data Is Processed

When you click "Review", "Describe", or "Explain":

  1. The extension extracts diff text from the current GitHub PR page
  2. The diff is sent to our Cloudflare Worker backend (or directly to the AI API if you use your own key)
  3. The AI generates a review and returns it to the extension
  4. No diff content is stored after the response is returned

Third-Party Services

  • Anthropic (Claude API) — Processes code review requests. Subject to Anthropic's Privacy Policy.
  • Lemon Squeezy — Handles payment and license validation. Subject to Lemon Squeezy's Privacy Policy.
  • Cloudflare Workers — Proxies API requests. No data is logged or stored.

Data Storage

All user settings (API key, license key, usage count) are stored locally in your browser using chrome.storage.local. No data is stored on our servers.

Permissions

  • storage — Save your settings locally
  • activeTab — Access the current tab to inject the review UI
  • https://github.com/* — Only activate on GitHub pages

Your Rights

You can delete all locally stored data at any time by removing the extension from Chrome. No server-side data exists to delete.

Changes to This Policy

We may update this policy as the extension evolves. Changes will be posted on this page with an updated date.
